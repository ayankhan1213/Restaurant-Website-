🍽️ Iqbal Nihari Restaurant Website — Authentic Taste of Traditional Desi Cuisine
Live Demo : https://heroic-florentine-da3712.netlify.app/

Iqbal Nihari Restaurant Website is a modern, responsive, and visually appealing web application designed to showcase the rich heritage and mouth-watering flavors of traditional Pakistani Nihari. This project represents a complete digital presence for a restaurant, built with a focus on elegant UI, smooth user experience, and clear presentation of menu items, services, and brand identity.

The website highlights the authentic taste of slow-cooked Nihari, special desi dishes, and premium dining experience offered by Iqbal Nihari. It is designed to attract food lovers by combining modern web design with traditional cultural aesthetics.

This project includes sections such as:

Home page with attractive hero section and restaurant branding
About section describing the restaurant’s history and specialty
Detailed menu showcasing Nihari variations and other dishes
Image gallery displaying high-quality food visuals
Contact section for reservations, orders, and location details

Built with performance and responsiveness in mind, the website works smoothly across all devices including mobile, tablet, and desktop. It is ideal for restaurant branding, online presence, and customer engagement.
